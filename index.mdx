## 文档分层

本仓库的文档按受众分为两类：

| 目录 | 受众 | 内容 |
| --- | --- | --- |
| `docs/` | 用户、外部开发者、包使用者 | 功能介绍、使用指南、价格说明、公开 API 和 SDK 文档 |
| `dev/` | 项目维护者 | 架构记录、任务清单、迁移计划、实现总结、测试计划和内部排障 |

根目录的 `docs/` 是 Mintlify 文档站，只配置用户可见内容。根目录和各子应用、子包的 `dev/` 不会被加入 Mintlify 导航。
